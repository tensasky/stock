## 1. 产品概述

### 1.1 产品定义
**白板红中**是由 AI 驱动的股票量化分析+模拟交易系统，通过技术指标筛选强势股票，自动执行买卖决策，并推送预警邮件。

### 1.2 核心目标
- 实现稳定量化投资收益
- 解放人工选股精力
- 严格执行交易纪律（止盈止损）

### 1.3 目标用户
- 散户投资者
- 量化交易爱好者
- 需要选股参考的投资者

### 3.2 选股策略 (V6.2)

**选入条件:**
```
✅ 量比: 1.2 - 3.0 (成交量放大但不过于极端)
✅ 多头排列: MA5 > MA10 > MA20
✅ 资金流入: 连续2日价涨量升
```

**评分体系:**
| 信号 | 分数 | 说明 |
|------|------|------|
| 多头排列 | +3 | MA5 > MA10 > MA20 |
| 资金流入 | +3 | 连续2日价涨量升 |
| 量比配合 | +2 | 量比1.2-3 |

**综合评分 ≥ 6分** → 触发买入信号

### 3.3 交易规则

| 规则 | 参数 | 说明 |
|------|------|------|
| 交易时段 | 9:35-11:30, 13:00-14:57 | 排除集合竞价 |
| 报告间隔 | 30分钟 | 每日盘中文易时段 |
| 止盈 | +5% | 达标后向上调整3% |
| 止损 | -2% | 触发即卖出 |
| 持有周期 | 2天 | 强制平仓窗口 |

### 3.4 决策流程

```
开盘扫描 (9:30-10:00)
    │
    ▼
获取观察池实时数据
    │
    ▼
计算技术指标 (MA, MACD, KDJ, RSI, 量比)
    │
    ▼
评分筛选 ──────▶ 分数 ≥ 6 ──▶ 买入信号
    │                         │
    │ No                      │
    ▼                         ▼
继续观察              更新持仓 + 发送邮件
    │
    ▼
盘中检查 (每30分钟)
    │
    ├── 止盈触发 (+5%) ──▶ 卖出
    ├── 止损触发 (-2%) ──▶ 卖出
    └── 持有2天 ─────────▶ 强制平仓
    
收盘报告 (14:30-15:00)
    │
    ▼
生成持仓报告
    │
    ▼
邮件推送 (洪福齐天)
```
