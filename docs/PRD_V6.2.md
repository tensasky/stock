## 3. 策略逻辑 (V6.2)

### 3.1 选股条件 (买入信号)

| 指标 | 分数 | 条件说明 |
|------|------|----------|
| **多头排列** | +3分 | MA5 > MA10 > MA20 |
| **资金流入** | +3分 | 连续2日收盘价上涨 AND 成交量上涨 |
| **量比配合** | +2分 | 1.2 < 当日量比 < 3.0 |

**总分**: 8分 (满分)

**买入门槛**: ≥8分

#### 详细说明

**多头排列**
- 计算20日内的MA5、MA10、MA20
- 短期均线在长期均线上方
- 代表上涨趋势形成

**资金流入**
- 连续2个交易日满足：
  - 收盘价 > 前一日收盘价 (价涨)
  - 成交量 > 前一日成交量 (量涨)
- 表示有资金持续买入

**量比**
- 量比 = 当日成交量 / 过去5日平均成交量
- 1.2-3区间：适度放量
- <1.2：成交量不足
- >3：放量过猛，可能放量下跌

---

### 3.2 卖出条件

| 条件 | 触发规则 |
|------|----------|
| **止损** | 亏损达到2%时卖出 |
| **止盈** | 盈利达到5%时卖出，达标后每上涨3%继续持有 |
| **时间止损** | 持有满2个交易日后强制卖出 |

---

### 3.3 技术指标 (辅助参考)

| 指标 | 计算方法 | 信号 |
|------|----------|------|
| **MACD** | EMA12 - EMA26 金叉/死叉 | 金叉买入，死叉卖出 |
| **KDJ** | (C-L9)/(H9-L9)×100 | K>80超买，K<20超卖 |
| **RSI** | 14日强弱指标 | RSI>70超买，RSI<30超卖 |

## 4. 评分机制

### 4.1 基础评分 (8分制)

```
总分 = 多头排列(3) + 资金流入(3) + 量比(2) = 8分
```

### 4.2 评级划分

| 得分 | 评级 | 操作建议 |
|------|------|----------|
| 8分 | S级 | 强烈买入 |
| 5-7分 | A级 | 可以买入 |
| 3-4分 | B级 | 观望 |
| 0-2分 | C级 | 不建议 |

### 4.3 高级指标加分 (可选)

| 指标 | 状态 | 加分 |
|------|------|------|
| MACD金叉 | 🔴 | +2分 |
| KDJ超卖 | K<20 | +1分 |
| RSI正常 | <70 | +1分 |

## 9. 使用流程

### 9.1 日常操作

```
1. 开盘前 (09:30前)
   └─> 运行 scan 扫描候选股票

2. 交易时段 (09:35-14:57)
   └─> 每30分钟运行 check 检查持仓
   └─> 根据信号决定买入/卖出

3. 收盘后 (15:00后)
   └─> 运行 report 发送日报
```

### 9.2 命令示例

```bash
# 扫描候选股票
python sim_trade.py scan

# 检查持仓
python sim_trade.py check

# 查看状态
python sim_trade.py status

# 发送报告
python sim_trade.py report

# 实时分钟均线
python realtime_ma.py
```

### 术语解释
- **MA**: 移动平均线 (Moving Average)
- **MACD**: 指数平滑异同移动平均线
- **KDJ**: 随机指标
- **RSI**: 相对强弱指标
- **量比**: 当日成交量 / 近5日平均成交量
