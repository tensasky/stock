# 🤖 股票信号分析系统 - V6 动态股票池

> 由 OpenClaw AI 开发的自动股票分析 + 邮件推送系统

---

## 📋 功能概述

1. **每日14:30自动分析** - 从观察池中筛选符合条件的股票
2. **动态股票池** - 根据技术信号自动选入/移除
3. **邮件推送** - 发送详细分析报告给指定收件人
4. **止盈止损跟踪** - 自动监控持仓股票

## 📊 V6 策略系统

### 选入条件
- **量比 > 2** (成交量放大)
- **至少2个技术信号**

### 技术信号评分

| 信号 | 分数 | 说明 |
|------|------|------|
| 多头排列 | +3 | MA5 > MA10 > MA20 |
| 资金流入 | +3 | 连续2日价涨量升 |
| MACD金叉 | +2 | DIF上穿DEA |
| MACD红柱 | +1 | MACD柱状图>0 |
| KDJ超卖 | +2 | K<20 或 J<0 |
| RSI超卖 | +2 | RSI<35 |
| 量比>2 | +2 | 成交量放大2倍 |
| 放量突破 | +2 | 量比>1.5且站上20日线 |

### 分类标准

| 分类 | 分数 | 操作建议 |
|------|------|----------|
| 强势 | ≥6分 | 可加仓 |
| 适中 | 4-5分 | 持有观望 |
| 观察 | <4分 | 等待机会 |

### 移除条件

1. **达到止盈** (+5%)
2. **跌破止损** (-3%)
3. **10个交易日无强势信号** (评分<4分)

## 📈 选入/移除记录

报告包含以下信息：

| 字段 | 说明 |
|------|------|
| 选入日期 | 何时被选入股票池 |
| 选入原因 | 量比多少 + 哪些技术信号 |
| 当前价格 | 实时价格 |
| 持有天数 | 已持仓交易日数 |
| PnL | 收益率 |
| 移除日期 | 何时被移除 |
| 移除原因 | 止盈/止损/超时 |

## 🚀 使用方法

### 1. 安装依赖
```bash
pip install pandas numpy
```

### 2. 配置邮件
修改 `mail_config.json` 中的：
- SMTP服务器和端口
- 发件人账号和授权码
- 收件人列表

### 3. 运行分析
```bash
cd stock_analyzer
python3 daily_v6.py
```

### 4. 设置自动运行
```bash
crontab -e
# 添加: 30 14 * * 1-5 /usr/bin/python3 /path/to/daily_v6.py
```
